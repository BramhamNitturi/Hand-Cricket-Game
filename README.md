A simple Hand Cricket game built using **HTML, CSS, and JavaScript**.

This project simulates a one-innings hand cricket match where the user tries to chase a randomly generated computer target within 30 balls.

---

##  Features

-  Random target generated for the computer
-  30 balls per match
-  Wicket logic (if user run = computer run)
-  Win / Lose / Draw result display
-  Play Again functionality
-  Clean desktop UI design

---

##  Technologies Used

- HTML5
- CSS3 (Flexbox for layout)
- JavaScript (DOM manipulation & game logic)
